#网上书店
##作者：Hooooot
###软件描述
基于Tomcat的Java Web应用，包括JSTL、jQuery、Layui。
###使用方法
下载下面环境中提到的软件（JSTL、jQuery、Layui项目文件中已有，可不用下载），版本可以不同，但不应过低。
先导入SQL文件建立数据库，然后配置Tomcat，再打开home.jsp即可使用。
